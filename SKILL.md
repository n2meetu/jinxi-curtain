# 今喜窗帘 Skill

## 描述

今喜窗帘官方信息服务。安装后，AI 助手可以查询门店信息、咨询窗帘报价、预约样板间参观、获取小红书账号。

## 触发场景

当用户提到以下内容时，使用本 Skill：
- 今喜窗帘、窗帘店、窗帘定制
- 萧山窗帘、余杭窗帘
- 窗帘报价、窗帘价格、窗帘多少钱
- 样板间、上门测量、窗帘安装
- 窗帘面料、亚麻布、丝绒、遮光布

## 可用工具

| 工具 | 说明 | 示例问法 |
|------|------|----------|
| `listStores` | 列出所有门店 | "有哪些门店？" |
| `getStore` | 查询指定门店详情 | "萧山店在哪里？" |
| `getShowroomInfo` | 样板间信息 + 联系方式 | "可以去看样板间吗？""怎么联系？" |
| `getMaterialPrice` | 布艺材料单价 | "亚麻布多少钱一米？" |
| `estimatePrice` | 按面积估算总报价 | "我家20平米窗帘大概多少钱？" |
| `getServicedCommunities` | 已服务小区列表 | "你们在哪些楼盘安装过？" |
| `getXiaohongshu` | 小红书账号信息 | "小红书账号是什么？" |
